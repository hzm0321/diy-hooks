# 简介
本章主要收录与生命周期相关 hooks

## 相关 hooks
- **[useEffectOnce](/life-cycle/use-effect-once)** 在组件的挂载和卸载阶段只执行一次。
- **[useMount](/life-cycle/use-mount)** 组件挂载阶段执行
- **[useUnmount](/life-cycle/use-unmount)** 组件卸载阶段执行
- **[useUpdate](/life-cycle/use-update)** 强制组件重新渲染
